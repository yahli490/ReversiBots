    Game rules

  • You should implement a bot that plays Reversi.
  • During the activity, two tournaments will be held on Wednesday (tentative). The first one will have each group's bot playing against three random bots, and the second one will be everyone vs. everyone.
  • The final score will be the amount of games won in both tournaments.
  • Upload your python code in the submit page. For your convenience you have been given an example bot.
  • Don't attempt to attack the server in any way, including DOS! while such attacks might be possible, we monitor your codes and will involve the program if it happens.
  • In order to practice, you can play against "naiveBot", "lessNaiveBot"
  • Due to a resources problem, you are given 0.6 seconds for each turn. If you surpass that, you will automatically lose the game.
  • Behind the scenes, in each turn a new instance of your code will be run. For that reason, don't keep information (global variables) between moves, as it will be lost.
